# 🎮 Programando com Jogos! — Rachel Maia

Landing page desenvolvida para divulgação de aulas de programação voltadas para crianças e adolescentes.

O projeto foi criado utilizando HTML, CSS e JavaScript puro, com foco em:

- Design moderno e amigável
- Responsividade para dispositivos móveis
- Conversão de leads via WhatsApp
- Exibição de projetos de alunos
- Estrutura simples para hospedagem no GitHub Pages

---

# 📚 Sobre o Projeto

O site foi desenvolvido para apresentar:

- Informações sobre a professora Rachel Maia
- Benefícios da programação para crianças
- Jogos e projetos criados pelos alunos
- Depoimentos de pais e alunos
- Cursos disponíveis
- Perguntas frequentes
- Formas de contato

A proposta do projeto é transformar tecnologia e programação em algo divertido, criativo e acessível.

---

# 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GitHub Pages

# 📱 Funcionalidades

## ✅ Landing Page Responsiva
O site se adapta para:

- Computadores
- Tablets
- Celulares

---

## ✅ Cards Interativos
Os cursos possuem expansão dinâmica utilizando JavaScript.

---

## ✅ FAQ Interativo
As perguntas frequentes utilizam sistema accordion.

---

## ✅ Vídeos Locais
Os vídeos dos alunos são hospedados diretamente no projeto utilizando a tag `<video>`.

---

## ✅ Integração com WhatsApp
Botões de contato direcionam diretamente para o WhatsApp.
